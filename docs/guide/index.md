# 入门指南

欢迎来到量化研究入门资料！本项目由湖南大学金融科技协会 Quant Group 维护。

## 项目简介

本项目旨在提供一套完整的多因子股票量化投研框架教程，涵盖：

- 量化金融、机器学习、数学参考书与资料
- 技术指标回测代码
- 卖方金工研报
- 投资者情绪与行为金融相关论文
- arXiv/SSRN 最新论文追踪
